Pour faire simple:

grep 'HotLikeBreizh' enc|awk -F ':' '{ print "_"$NF"="$3 }' |tr -d '^' |xargs echo  |tr ' ' ';'

Ca transforme chaque ligne ‘8====D::^::#HotLikeBreizh::92’ en ‘$_92=’. Il faut rajouter un underscore avant chaque valeur car bash n’accepte pas une variable qui commence avec un chiffre.

Ensuite :

echo -n "echo "; grep ^# enc |tr -d '#' |sed 's/\(..\)/$_\1/g'

on transforme la ligne 8c15e8… en:

$_8c$_15$_e8
